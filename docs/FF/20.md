??? note "JSON"

    ``` javascript
    // console 输入，输出错误 "Uncaught SyntaxError: Unexpected token ' in JSON at position 1"
    JSON.parse(`{'name':'frank'}`)
    ```

    ``` javascript
    let object
    try {
        object = JSON.parse(`{'name':'frank'}`)
    } catch(error) {
        console.log('出错了，错误详情是')
        console.log(error)
        object = {'name':'no name'}
    }
    console.log(object)
    ```

    ``` javascript
    // console
    > const obj = { "hi":'ho', fn: ()=>{} }
    < undefined
    > JSON.stringify(obj) 
    < "{"hi":"ho"}"
    ```
