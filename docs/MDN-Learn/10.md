## 3) web server

* An HTTP server: 

    > 一个软件：理解 URLs (web addresses) 和 HTTP (the protocol your browser uses to view webpages).

* A static web server:

    > 电脑 (硬件) + an HTTP server (软件)

* A dynamic web server

    > a static web server + 额外的软件, 
    > 
    > 最常见的是 an application server and a database
