    JS 函数的执行时机¶

    1 解释为什么如下代码会打印 6 个 6

    let i = 0
for(i = 0; i<6; i++){
  setTimeout(()=>{
    console.log(i)
  },0)
}

    setTimeout() 的延迟时间的值为 0，意思是"immediately"执行，或者更准确地说，the next event cycle（事件周期）。

    当执行 console.log(i) 时，i 的值变为 6 而且循环积累了 6 个函数待执行，所以打印出的值是 6 个 6。

    2 写出让上面代码打印 0、1、2、3、4、5 的方法

    for (let i = 0; i<6; i++){
    setTimeout(()=>{
        console.log(i)
    },0)
}

    let 语句声明 块级作用域 的变量，和前面 for 循环使用全局变量 i 不同，当前的 i 只在本轮循环有效。

    3 除了使用 for let 配合，还有什么其他方法可以打印出 0、1、2、3、4、5

    1. 用 立即执行函数 得到 局部变量 i

      let i = 0
for(i = 0; i<6; i++){
    setTimeout(!function(){
        console.log(i)
    }(),0)
}

    2. 利用 setTimeout 的第三个参数,将i传进去

      let i = 0
for(i = 0; i<6; i++){
    setTimeout((i)=>{
        console.log(i)
    },0,i)
}
